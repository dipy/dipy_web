
==================
dipy_labelsbundles
==================

Usage
-----

 dipy_labelsbundles [-h] [--out_dir str] [--out_bundle str] streamline_files labels_files

Extract bundles using existing indices (labels)

Positional Arguments
--------------------


  streamline_files  The path of streamline files where you want to recognize bundles.
  labels_files      The path of model bundle files.

Optional Arguments
------------------


  -h, --help        show this help message and exit

Output Arguments(Optional)
--------------------------


  --out_dir str     Output directory. (default current directory)
  --out_bundle str  Recognized bundle in the space of the model bundle.
